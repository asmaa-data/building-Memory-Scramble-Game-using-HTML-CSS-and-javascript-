Memory Scramble Game
 ====================
A modern Memory Matching Game developed using:
- HTML
- CSS
- JavaScript
The player flips cards and tries to match all pairs before the countdown timer reaches zero.


Features
=========
Dynamic board size configuration  
Countdown timer  
Random card shuffle  
Win and Game Over Alerts  
Friendly UI with animations  
Responsive and interactive design  


Rules
========
1. Player Should Enter:
   - Number of rows
   - Number of columns
   - Time limit

2. The board size must be EVEN.

3. Click "Start Game".

4. Flip two cards:
   - If they match → they stay open.
   - If they do not match → they flip back.

5. Match all pairs before the timer ends to win.
